## Questions Judges Might Ask

**Q: How is this different from regular cloud storage?**  
A: "Client-side encryption means the server never sees plaintext. Plus blockchain verification provides immutable proof of storage."

**Q: What about encryption key management?**  
A: "Keys are generated in TEE secure enclaves. Users can recover via social recovery or hardware wallets."

**Q: How do AI agents access the memory?**  
A: "Via our SDK—agents request access, user approves, credentials are verified on-chain, encrypted data is fetched and decrypted client-side."

**Q: What's the cost?**  
A: "Base Sepolia gas is minimal—about $0.50 per memory store. Filecoin storage is competitive with traditional cloud."
